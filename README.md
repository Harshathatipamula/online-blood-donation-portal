# 🩸 Online Blood Donation Management System

A full-stack web application that connects blood donors with recipients.  
The system allows users to register as donors, search blood availability by group and location, and manage blood donation requests efficiently.

This project aims to simplify the blood donation process and provide quick access to blood donors during emergencies.

---

# 🚀 Features

- Donor Registration
- Blood Group Search
- Location-based donor availability
- Blood Request Management
- Secure backend API
- Responsive user interface

# ⚙️ Installation Guide

1️⃣ Clone the repository
git clone https://github.com/Harshathatipamula/online-blood-donation-portal.git


2️⃣ Navigate to project folder
cd online-blood-donation-portal



3️⃣ Install backend dependencies
npm install


4️⃣ Start the server
node server.js


5️⃣ Open the frontend in browser
http://localhost:5000

# 🔗 API Endpoints

| Method | Endpoint | Description |
|------|---------|-------------|
| POST | /register | Register a new donor |
| GET | /donors | Get all donors |
| GET | /donors/:bloodgroup | Search donors by blood group |
| POST | /request | Create blood request |
